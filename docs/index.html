DevText CLI

Format JSON, XML, YAML, and strip markdown code blocks — from the command line

View on GitHub Install Now

Install

npm install -g devtext-cli

Or use npx (no install required):

npx devtext-cli --help

Features

JSON

Pretty-print, validate, minify. Works with stdin, files, or piped output.

echo '{"key":"value"}' | devtext -f json

XML

Format and validate XML documents. Perfect for legacy API responses.

devtext data.xml -f xml

YAML

Convert YAML to JSON. Great for Kubernetes configs and CI/CD files.

devtext config.yaml -f yaml

Strip Markdown

Remove code blocks from AI outputs. Works with ChatGPT/Claude responses.

echo '```json {"test":1} ```' | devtext -f strip
